How to run this Group 2 E-Commerce Website User Registration & Login System

1. Download the zip file

2. Extract the file

3. Create a new folder named "CrestStore"

4. Move the extracted files into the "CrestStore" folder

3. Move the "CrestStore" folder into your root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name "creststoredb"

6. Import creststore.sql file (given inside the zip package in SQL file folder)

7. Run the script http://localhost/creststore/users/home.php
